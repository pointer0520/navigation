# 导航网站使用指南

## 🚀 快速开始

### 1. 访问登录页面

前端已启动，访问地址：`http://localhost:3000`

由于添加了登录验证，首次访问会自动跳转到登录页面。

### 2. 注册新用户

**步骤：**
1. 在登录页面点击"还没有账号？立即注册"
2. 填写注册信息：
   - **用户名**：3-20个字符
   - **密码**：至少6位
   - **邮箱**：有效的邮箱地址（可选）
3. 点击"注册"按钮
4. 注册成功后会自动登录并跳转到首页

**示例账号：**
```
用户名: testuser
密码: 123456
邮箱: test@example.com
```

### 3. 登录系统

**步骤：**
1. 输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后跳转到首页

**测试账号：**
- 使用你刚刚注册的账号登录
- 或者通过后端 API 直接创建测试账号

## 📝 功能使用

### 创建分类

1. **进入首页**
   - 登录后自动进入首页

2. **新建分类**
   - 点击左侧导航底部的"新建分类"按钮
   - 填写分类信息：
     - **名称**：例如"开发工具"
     - **图标**：选择一个图标
     - **颜色**：选择颜色
     - **父分类**：可选，用于创建子分类
   - 点击"确定"

3. **查看分类**
   - 新建的分类会立即显示在左侧导航栏
   - 点击分类可以查看该分类下的站点

### 添加站点

1. **点击添加按钮**
   - 右上角点击"添加站点"按钮

2. **填写站点信息**
   - **站点名称**：例如"GitHub"
   - **网址**：例如"https://github.com"
   - **图标**：会自动获取，也可手动输入
   - **分类**：可多选
   - **标签**：可多选或创建新标签
   - **快速访问**：开关控制是否加入快速访问

3. **提交**
   - 点击"确定"按钮
   - 站点会显示在对应分类下

### 搜索站点

1. **使用搜索框**
   - 顶部搜索框输入关键词
   - 按回车或点击搜索按钮

2. **查看结果**
   - 跳转到搜索结果页
   - 显示匹配的站点列表

### 快速访问

- 开启"快速访问"的站点会显示在首页顶部
- 点击可快速打开站点
- 自动统计访问次数

## 🔐 账户管理

### 查看用户信息

- 点击右上角用户头像图标
- 下拉菜单显示当前用户名

### 退出登录

1. 点击右上角用户头像
2. 选择"退出登录"
3. 自动跳转到登录页面

### 重新登录

- 退出后可以重新登录
- Token 有效期为 24 小时
- Token 过期会自动跳转到登录页

## 🎨 界面操作

### 分类导航

**展开/折叠：**
- 点击导航栏顶部的折叠按钮
- 折叠后只显示图标

**切换分类：**
- 点击分类查看对应站点
- 再次点击取消选择，显示全部站点

### 视图切换

**网格视图 / 列表视图：**
- 站点列表右上角切换按钮
- 网格：卡片式展示
- 列表：详细信息展示

### 站点操作

**访问站点：**
- 点击站点卡片在新标签页打开

**编辑站点：**
- 鼠标悬停显示编辑按钮
- 点击编辑图标修改信息

**删除站点：**
- 鼠标悬停显示删除按钮
- 点击删除图标确认删除

## 🐛 常见问题

### Q1: 访问首页自动跳转到登录页
**A:** 这是正常的，系统添加了登录验证。请先注册/登录。

### Q2: 登录后提示"加载数据失败"
**A:** 
1. 检查后端服务是否启动
2. 确认后端运行在 `http://localhost:8080`
3. 查看浏览器控制台错误信息

### Q3: 注册时提示"用户名已存在"
**A:** 换一个用户名重新注册，或使用现有账号登录。

### Q4: Token 过期怎么办？
**A:** 
- Token 有效期 24 小时
- 过期后会自动跳转到登录页
- 重新登录即可

### Q5: 分类和站点数据不显示
**A:** 
1. 确认已登录
2. 检查后端服务状态
3. 查看 Network 标签的 API 响应
4. 确认数据库中有数据

## 📊 数据同步

### 本地存储 vs 云端同步

- **本地模式**：数据存储在 localStorage（已废弃）
- **云端同步**：所有数据存储在 MySQL 数据库
- **实时同步**：操作立即同步到服务器

### 多设备使用

1. 在不同设备使用相同账号登录
2. 数据自动同步
3. 添加/修改/删除操作实时生效

## 🔑 安全建议

1. **密码安全**
   - 使用强密码（至少 8 位，包含字母数字）
   - 定期更换密码

2. **数据备份**
   - 定期导出数据（功能待实现）
   - 保存重要站点信息

3. **隐私保护**
   - 不要在公共电脑保存登录状态
   - 使用完毕及时退出登录
