# 个人导航站 - 开发指南

## 📋 项目概述

这是一个基于需求文档分阶段开发的个人导航站Web项目，采用现代化的前端技术栈构建。

## 🎨 核心数据模型

### Site（站点实体）
```typescript
interface Site {
  id: string              // 唯一标识
  name: string            // 站点名称
  url: string             // 网址
  iconUrl?: string        // 图标URL
  addTime: number         // 添加时间戳
  visitCount: number      // 访问次数
  categoryIds: string[]   // 所属分类ID列表
  tagIds: string[]        // 关联标签ID列表
  isQuickAccess: boolean  // 是否快速访问
}
```

### Category（分类实体）
```typescript
interface Category {
  id: string              // 唯一标识
  name: string            // 分类名称
  icon: string            // 图标名称
  color: string           // 颜色值
  parentId?: string       // 父分类ID
  sortOrder: number       // 排序序号
}
```

### Tag（标签实体）
```typescript
interface Tag {
  id: string              // 唯一标识
  name: string            // 标签名称
  color: string           // 颜色值
  createTime: number      // 创建时间戳
}
```

## 📱 功能使用指南

### 添加站点
1. 点击顶部"添加站点"按钮
2. 输入站点名称和URL
3. 系统自动获取网站图标
4. 选择分类和标签（可多选）
5. 勾选"快速访问"以加入首页
6. 点击"确定"保存

### 管理分类
1. 进入"设置" > "分类管理"
2. 点击"新建分类"
3. 输入名称、选择图标和颜色
4. 可设置父分类创建二级分类
5. 拖拽分类项调整排序
6. 点击编辑/删除按钮管理分类

### 搜索站点
1. 在顶部搜索框输入关键词
2. 使用筛选器按分类/标签过滤
3. 选择排序方式
4. 点击结果直接访问站点
5. 搜索历史自动记录

### 数据备份
1. 进入"设置" > "数据备份"
2. 点击"导出数据"下载JSON文件
3. 使用"导入数据"上传备份文件
4. 支持清空所有数据重新开始
